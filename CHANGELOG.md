# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-02-11
- Changed module ID from `custom-5e-tools` to `tools-proficiency-5e`.
- Updated release artifact name to `tools-proficiency-5e.zip`.
- Updated manifest `download` URL to the new release asset name.

## [1.0.4] - 2026-02-11
- Renamed the public module title to "Tools Proficiency 5e" to reduce naming confusion.
- Kept module ID as `custom-5e-tools` for backward compatibility and update continuity.

## [1.0.3] - 2026-02-11
- Fixed custom Other tools to merge into dnd5e's existing Other section.
- Fixed duplicate custom key suffix format to avoid parser issues.

## [1.0.2] - 2026-02-11
- Added actor data repair pass for malformed custom tool entries.
- Switched custom reference IDs to synthetic module IDs.

## [1.0.1] - 2026-02-11
- Changed custom tool slug format to avoid invalid proficiency submissions.

## [1.0.0] - 2026-02-10
- Initial release.
- Added configurable custom tool lists by category.
- Injected custom tools into dnd5e tool proficiency UI.
